Папка FOOTER и HEADER - одноимённые компоненты из index-ов со своими стилями

MAIN_ASSETS - общие для всех сайтов стили и js

ITEM_CATALOG - общие для карточек товаров js, css и структура index,
		в assets-img нужные изображения для данного index

MENI_CATALOG - общие для страниц карточек каталога css,
		отдельные папки со своими index-ами, css и img

THANKS - папка со страницей благодарности за заказ

В js файлах есть задатки для БД
